Adidas Finale 13 - Match Ball

Creating a bespoke set of prints for an iconic global brand

Client Football Adidas
Role Illustrator

Each year Adidas produces a ‘Finale' ball for use in both the knock-out stages and the final match of the UEFA Champions League. The ‘Wembley 13′ was the 8th such ball to be released and I was asked to create a series of prints that celebrated each of the previous balls together with the location of those specific finals.

"Steve produced a bespoke set of artwork for Adidas which we subsequently promoted across our global football social media platforms. I’m happy to recommend Steve’s talents – his work rate and attention to detail was first class throughout and this was something which came across strongly in the final prints".

Alan McGarrie — Global PR Manager for Football Adidas
